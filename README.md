# Family Meal Planner

A 14-day rotating meal planner with smart variety constraints. Runs as a Progressive Web App — open the URL on your phone, tap Share → Add to Home Screen, and use it like a native app.

## Files

- `index.html` — the app (self-contained: HTML, CSS, JS)
- `manifest.json` — PWA manifest (app name, icon, display mode)
- `icon-192.png`, `icon-512.png` — Android home-screen icons
- `apple-touch-icon.png` — iOS home-screen icon

## Hosting

Deployed via GitHub Pages. Data lives in each device's browser localStorage — no backend.
